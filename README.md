# RESTfull-API

Постановка задачи

Создать RESTful API для управления библиотекой. Сервис работает с тремя моделями: Авторы, Книги и Читатели.

Модели:

Автор (Author)

id - целочисленный уникальный идентификатор
name - строка до 100 символов (ФИО)
birth_date - дата рождения
country - строка до 50 символов (страна)
Книга (Book)

id - целочисленный уникальный идентификатор
title - строка до 200 символов
author_id - id автора (связь с автором)
year - год издания (от 1000 до 2024)
genre - строка до 50 символов
is_available - булево значение (доступна ли книга)
Читатель (Reader)

id - целочисленный уникальный идентификатор
name - строка до 100 символов
registration_date - дата регистрации
email - строка (email читателя)
books_taken - список id взятых книг
